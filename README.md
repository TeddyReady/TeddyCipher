# TeddyCipher
Text Cipher from cyrillic based on Qt
## Building
1. Clone this repo:

        git clone https://github.com/TeddyReady/TeddyCipher.git && cd TeddyCipher

2. Create a build directory:

        mkdir build && cd build

3. Run qmake && make:

        ../qmake && make -j4
        
4. Run application:
    
        ./TeddyCipher
